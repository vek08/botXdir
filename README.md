# BotXDir – AI Chatbot GUI Application

BotXDir is a desktop-based AI chatbot application built using Python and Tkinter.  
The application supports both text and voice interaction, providing an interactive conversational experience with speech recognition and text-to-speech capabilities.

## Features

- AI-powered chatbot responses using BlenderBot
- Voice input using SpeechRecognition
- Text-to-speech response system
- Desktop GUI built with Tkinter
- Real-time conversational interaction
- Multi-threaded voice handling
- Clean and interactive user interface

## Technologies Used

- Python
- Tkinter
- Transformers (Hugging Face)
- BlenderBot
- SpeechRecognition
- pyttsx3
